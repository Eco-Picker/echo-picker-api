# Eco Picker API

## Prerequisites

- JDK 17 or higher
- Gradle
- IDE (IntelliJ IDEA recommended)

## Getting Started

### git clone

```shell
git clone https://github.com/Eco-Picker/eco-picker-api.git
cd eco-picker-api
```

### set an env file

```shell
cp .env.exmaple .env
```

### bootRun

```shell
./gradlew bootRun
```

### Swagger

http://localhost:15000/swagger-ui/index.html
